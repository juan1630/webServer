# Primer Webserver con node js 

## Es un web server para el curso de node js

### Se uso: 

- Express
- Rutas con express
- Handlebars
- Interpolacion de variables entre node js y handlebars
- Helpers de hbs
- Partilas en hbs

#### Comandos 


``` 
npm install
```
``` 
npm install --save hbs
```
``` 
npm install --save express
```
